Welcome to LSTMusic!

LSTMusic is a site I've created to allow for accessible, playful experimentation with machine-learning and music.

This site is intended to be a fun and engaging tool for anyone interested in music and machine-learning,
so please don't take it too seriously as a music production tool.

Using LSTMusic

To get started, follow these steps:

1. Input a melody by clicking and dragging to add notes, and right-clicking selected notes to delete them.
2. Adjust the generator's adventurousness, extension length, and tempo to your liking.
3. Click the "Expand!" button to generate a longer melody after a short wait.
